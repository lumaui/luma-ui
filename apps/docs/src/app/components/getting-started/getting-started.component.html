Getting Started

Three simple steps to start building with Luma UI

1

Prerequisites

Set up Tailwind CSS v4 in your Angular project first.

View Tailwind guide
2

Installation

Install the package via npm.

npm install @lumaui/angular

Dependencies are installed automatically.

3

Setup

Import tokens in your styles.css 

@import 'tailwindcss';
@import '@lumaui/tokens';
@import '@lumaui/tokens/dark.css';
